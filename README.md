🚀 How to Use
Input: You can enter numbers and operators using either your mouse (clicking the on-screen buttons) or your keyboard.

Calculate: To get the result, press the Enter key on your keyboard or click the = button on the calculator.

Delete & Clear: * Press Backspace to delete the last entered character.

Press Delete on your keyboard or click the AC button on the screen to completely clear the input.

⚠️ Important Note on Performance:
The backend of this application is deployed on a free hosting tier. If the application hasn't been used for a while, the server goes into a "sleep" mode to save resources. When you make your very first calculation, it may take about a minute for the backend to wake up and process the request. Please be patient! After this initial delay, all subsequent calculations will be processed instantly.
